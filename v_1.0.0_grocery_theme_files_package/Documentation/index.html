1.2 Install with Template Package:

Upload Theme Files

After you complete the installation of opencart, please follow below steps to install themes

  • Step 1 - Unzip the Grocery_template_install.zip package.
Replace Theme Files
  • Paste these directories from 'Theme-package' to your opencart root folder.
  • "admin" (Overwrite this folder)
  • "catalog" (Overwrite this folder)
  • "image" (Overwrite this folder)

AFTER UPLOAD THEME FILES, YOU MAY GET ERROR(COULD NOT LOAD FILE) ON FRONT SIDE.
BUT DON'T WORRY, WHENYOU ACTIVE OUR THEME, IT IS WORK PERFECTLY.

After Completing Installation Steps Must be Select Theme Store In Admin Panel for that:
  • For Default Home : Go to Extensions > Extensions > Themes > default > Edit

Step2 : Logo and Image size settings

  • For Logo setting , Go to System > Setting > Edit > Images,
  • From Store Logo you can upload your logo image.
For Image Size setting ,
For Default_Home : Go Extensions > Extensions > Themes > Destino > Edit ,
  • Change category image thumb size to: 80 X 80,
  • Change product image thumb size to: 568 X 568,
  • Change Product Image Popup Size to: 700 X 700,
  • Change product image list size to: 700 X 700 ,
  • Change additional product image size to: 124 X 124,
  • Change related product image size to: 700 X 700 ,
  • Change wish list image size to: 47 X 47 ,
  • Change cart image size to: 47 X 47 ,
  • Change store image size to: 268 X 50 .
  • Click on “Save” button.
  • For Store setting Go To System->Settings Click on store Edit link, and set below settings.

Step 3 : Language Settings

To Enable Laguages login to your admin panel and do follwing settings.

  • For Example To add Arabic Language :
  • Go to System > Localisation > Languages,
  • Click on “Add New” button to add new Language.
  • Language Name : Arabic,
  • Code : ar,
  • Locale : ar-UN,ar_UN,UTF-8,Arabic,
  • Status : Enable,
  • Sort Order : as you like.
  • Click on “Save” button.

Step4 : Theme Layouts

4.1.1. Module : Slideshow->Home Page Slideshow

  • For adding Slideshow banners go to Design > Banners
  • Click on “Add New” button to add new banner.
  • Give Banner name: Home Page Slideshow,
  • From status select Enabled,
  • and click on “Add Banner” button.
  • Give title of your banner and click on image to upload your banner.
  • Again click on “Add Banner” button and upload another banners.
  • Now click on “Save” button.
  • Now to activate your Main banner go to Extensions > Extensions > Modules > Slideshow click on Edit link
  • Give Module name: Home Page Slideshow ,
  • From banner select Home Page Slideshow,
  • Give Width & Height (575 X 450) (your banner Width and height),
  • From status select Enabled,
  • Now click on “Save” button.
